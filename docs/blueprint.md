# **App Name**: StackFund: Bitcoin Crowdfunding

## Core Features:

- Discover Projects: Display a grid of active crowdfunding campaigns with key details and progress indicators.
- Campaign Details: Show complete information for a selected campaign, including funding progress, countdown timer, and creator details.
- Fund Project: Allow users to contribute sBTC to a campaign using Turnkey Embedded Wallet SDK for secure and seamless transactions.
- Create Campaign: Enable users to create and launch their own crowdfunding campaigns by filling out a simple form with project details.
- Wallet Dashboard: Provide a wallet interface displaying user's sBTC balance, transaction history, and created campaigns.
- Campaign Storage: Optionally leverage Supabase/Firebase for persistent storage of campaign data.

## Style Guidelines:

- Primary color: Bitcoin gold (#F7931A) for a strong association with Bitcoin and digital finance.
- Background color: Warm off-white (#FAFAF8), offering a clean and modern look that promotes readability.
- Accent color: Soft green (#22C55E), providing visual cues for success states and positive actions within the platform.
- Body and headline font: 'Inter', a sans-serif with a modern, neutral appearance suitable for a professional platform. Note: currently only Google Fonts are supported.
- Utilize simple and modern icons for clear communication and improved user experience. Icons should be related to finance, blockchain, and community engagement.
- Employ a minimal layout with rounded corners and light gradients to create a modern and trustworthy user interface.
- Implement subtle animations using Framer Motion to enhance user engagement, such as hover effects, progress bar animations, and transaction confirmations.
